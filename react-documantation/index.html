Introduction

INTRODUCTION

CryptWallet is a modern cryptocurrency template based on React Boilerplate and Material ui Framework. We’ve used modern technologies and best practices to make our product easy to work with. It’s the most convenient template for developers, because of React Components, clean code and detailed documentation, which allows you to build any projects easily! Use it for e-commerce, analytics and other types of web or mobile applications.
